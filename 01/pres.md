## if-then-else

`f x y = if x >= y then x else y`

Was macht `f`?
\pause
Das Maximum von `x,y` berechnen.

## switch-case

\pause

~~~
max3 x y z
  | x >= y && x >= z = x
  | y >= x && y >= z = y
  |otherwise = z
~~~

\pause
Der erste passende Fall wird ausgewertet.

Diese Struktur wird guards genannt.

## Pattern matching

~~~
summiere [] = 0
summiere (x:rest) = x + (summiere rest)
~~~

Zerteilung der Liste in Kopf und Rest.
